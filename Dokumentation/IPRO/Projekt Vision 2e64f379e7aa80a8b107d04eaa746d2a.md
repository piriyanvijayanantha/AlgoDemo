# Projekt Vision

**Was?** Eine Java-Desktop-Anwendung mit grafischer Oberfläche

**Für wen?** Studenten die das Modul Prog1 besuchen. 

**Kontext?**  im Selbststudium für das Verständnis und Visualisierung von 1-2 Algorithmen die im Modul behandelt werden

**Tätigkeiten?** 

- Algorithmen Schritt für Schritt ausführen
    - Undo ein Schritt
- Variablenwerte manuell einfügen (i,j,m)
- Quiz fragen wie ( Welche zwei Werte werden als nächstes Verglichen?, An welchem Index wird die Variable m sein nach 3 Schritten?) unnötig???
- Automatische Schritte mit Geschwindigkeit
- Eigene Arrays Bestimmen

**Algorithmen für Prog1**

- Binary Search (Prio 1)
- Merge Sort (Prio 2)
